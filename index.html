📚 Panduan Menggunakan GitHub Pages

🚀 Cara Deploy ke GitHub Pages

  1. 1
    Buat Repository GitHub

    Login ke GitHub → New Repository → Beri nama (contoh: "daftar-nilai")

  2. 2
    Upload File HTML

    Upload file ini dengan nama index.html

  3. 3
    Aktifkan GitHub Pages

    Settings → Pages → Source: Deploy from branch → Branch: main → Save

  4. 4
    Akses Website

    URL: https://username.github.io/repository-name

✨ Fitur Sistem Online

  • ✅ Akses dari mana saja dengan internet
  • ✅ Data tersimpan di browser (LocalStorage)
  • ✅ Export/Import Excel untuk backup
  • ✅ Responsive - bisa di HP/Tablet
  • ✅ Gratis selamanya di GitHub Pages
  • ✅ SSL Certificate otomatis (HTTPS)

💡 Tips Penggunaan

  • 💾
    Backup Rutin:

    Gunakan tombol "Backup Data" untuk menyimpan data ke file

  • 🔄
    Sinkronisasi:

    Data tersimpan per browser, gunakan backup untuk pindah device

  • 📱
    Mobile Friendly:

    Bisa diakses dari smartphone untuk input cepat

🔧 Troubleshooting

  • ❌
    Data Hilang:

    Restore dari file backup atau clear browser cache

  • 🌐
    Website Tidak Muncul:

    Tunggu 5-10 menit setelah deploy, atau cek Settings → Pages

  • 📱
    Tampilan Mobile:

    Scroll horizontal untuk tabel besar, atau rotate ke landscape
